# 用Claude写商业文案

你写文案需要3小时。

Claude写只需要3分钟。

质量比你写的还好。

这就是AI时代的现实。
你不需要成为更好的写手。
你需要成为更好的提示工程师。

## 为什么选Claude

ChatGPT很火。
但写商业文案，Claude更擅长。

原因：
- 更长的上下文（200K token）
- 更自然的语气
- 更少的"AI味"

Vigma的AI搞钱手册里，Claude是首推的文案工具。

## 三步写出高转化文案

**第一步：喂素材**

不要直接说"帮我写个销售文案"。

给Claude这些：
- 你的产品介绍
- 目标客户画像
- 竞品文案（你喜欢和不喜欢的）
- 你想要的语气（幽默/专业/亲切）

素材越丰富，输出越精准。

**第二步：给框架**

告诉Claude结构：

```
按这个结构写：
1. 痛点开场（1句）
2. 反转（1句）
3. 产品介绍（3个卖点）
4. 社会证明（1个案例）
5. 行动号召（1句）

语气：像朋友聊天，不用专业术语
字数：300字以内
```

框架越清晰，改稿次数越少。

**第三步：迭代优化**

第一版不会完美。
用这些问题优化：

- "开头不够抓人，重写"
- "太像广告了，再自然一点"
- "加一些具体数字"
- "结尾太弱，来个有力的收尾"

Claude会记住上下文。
你可以像指导实习生一样指导它。

## 实战案例

假设你卖一个时间管理课程。

**你的提示词：**

```
我要写一个销售文案，推广我的时间管理课程。

产品信息：
- 帮助自由职业者提高工作效率
- 包含5个模块，每个30分钟
- 价格199元
- 已有500+学员，平均效率提升40%

目标客户：
- 自由职业者，收入不稳定
- 经常加班，感觉时间不够用
- 想提高收入但不知道怎么做

参考这个语气：
"你不是缺时间。你是缺方法。"

按AIDA模型写：
Attention（注意）- Interest（兴趣）- Desire（欲望）- Action（行动）
```

**Claude输出：**

```
你每天都在忙。

但月底看账单，收入还是那么多。

真相是：你不是缺时间。你是缺方法。

我花了3年，测试了几十种时间管理技巧。
最后提炼出一套适合自由职业者的方法。

5个模块。
每模块30分钟。
学完立即能用。

500+学员验证过。
平均效率提升40%。

也就是说：
你每天能省出3小时。
一年多赚10万+。

投资199元。
回报是10万。

这账，你会算。

[立即报名]
```

不错吧？

## 进阶技巧

**技巧一：角色扮演**

"你是一位有10年经验的营销专家，擅长写高转化文案。"

给Claude一个身份，输出质量会提升。

**技巧二：多版本对比**

"给我3个不同版本：
1. 专业版（面向企业客户）
2. 亲切版（面向个人用户）
3. 激进版（面向年轻群体）"

一次提示，多个选择。

**技巧三：反向工程**

把你喜欢的文案喂给Claude：
"分析这篇文案为什么有效。然后用同样的结构，为我的产品写一版。"

学习优秀案例，快速上手。

## 常见错误

**错误一：提示词太模糊**

❌ "帮我写个好文案"
✅ "帮我写一个面向25-35岁职场女性的护肤品文案，突出天然成分，语气亲切自然，300字"

**错误二：一次要太多**

❌ "写文案+设计图片+做视频脚本"
✅ 一次一个任务，分步骤来

**错误三：不检查事实**

Claude会编造数据。
所有数字、案例、引用，都要人工核实。

## 你的新工作流

3. A/B测试不同版本
4. 根据数据反馈，让Claude优化
