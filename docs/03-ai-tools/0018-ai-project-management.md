# 用AI做项目管理：一人公司也能井井有条

你是一个人的公司。
但你同时是产品经理、开发者、设计师、市场、客服、财务。

没有项目经理帮你排优先级。
没有协调人帮你跟进任务。
你只能靠自己。

这就是为什么你需要AI。

## 一人公司的项目管理问题

不是你不知道要做什么。
是你同时有20件事要做，不知道先做哪件。

客户A的需求还没交付。
课程B的录制还差3节。
文章C的草稿还躺在文档里。
 taxes D的截止日期快到了。

你的脑子不是项目管理工具。
它装不下这么多并行任务。

## AI怎么帮你管项目？

**1. 拆解任务**

把一个大目标扔给ChatGPT："我要在30天内上线一个课程，帮我拆解成每日任务。"

它会给你一个清晰的执行计划。
不是模糊的方向。是具体的每日行动。

**2. 排优先级**

把你今天的10个待办丢给AI："按紧急/重要矩阵帮我排序，并告诉我哪些可以不做。"

你会发现，至少一半的事情可以不做。
少做 = 更专注 = 更高效。

**3. 时间估算**

AI能帮你估算每件事需要多久。
你只需要诚实地说出你的速度。
然后AI帮你排出一个实际可行的日程。

**4. 进度追踪**

每天花5分钟，告诉AI你做了什么、卡在哪里。
它会帮你调整计划，重新排序，保持你的节奏。

这不是取代Notion或Todoist。
这是让AI成为你的项目搭档。

## 推荐的AI + 项目管理工作流

1. **每周一**：用AI生成这周的任务清单和优先级
2. **每天早上**：花3分钟和AI确认今天的Top 3
3. **每天晚上**：花2分钟和AI复盘今天的进度
4. **每周五**：用AI做周复盘，调整下周计划

Vigma体系中的"7天验证法"就是这种思路的极致应用——把每个新想法限定在7天内验证，用AI帮你拆解验证步骤，第7天做go/no-go决策。

## 实际操作：一个对话模板

你可以直接复制这个prompt：

> 我是一个一人公司创业者。以下是我当前的所有项目和待办：
> [列出所有事项]
> 
> 请帮我：
> 1. 按优先级排序
> 2. 标出可以删除或委派的
> 3. 生成本周的执行计划
> 4. 指出我可能忽略的风险

每周用一次。你会惊讶于AI的判断力。

## 常见问题

**Q: AI不会了解我的业务细节怎么办？**
你越具体地描述你的项目和目标，AI的输出越有用。垃圾进，垃圾出。详细进，精准出。

**Q: 需要专门的项目管理工具吗？**
轻量级用Notion或Obsidian就够了。AI是大脑，工具是笔记本。两者配合最好。

**Q: 会不会太依赖AI？**
AI是顾问，不是老板。最终的决策在你。AI帮你看到全貌，你来做判断。

## 一人公司最大的效率杀手

不是工具不够。
不是能力不行。
是你的任务没有系统。

脑子里的想法不是系统。
写在纸上的清单不是系统。
能追踪、能调整、能复盘的流程，才是系统。

AI帮你建系统。你负责执行系统。

管好项目，才能管好生意。
管好生意，才能管好生活。
