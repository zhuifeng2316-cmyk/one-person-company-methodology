# 用AI做客服

你一个人做公司。

客户咨询越来越多。
回复消息占用你一半时间。

你试过请兼职客服。
培训成本高，流动性大。

试试AI客服。
不是替代你，是放大你。

## AI客服能做什么

**即时回复**
24小时在线，秒级响应。
客户不用等，体验更好。

**常见问题解答**
产品功能、价格、使用方法。
80%的问题都是重复的。

**售前咨询**
了解客户需求，推荐合适产品。
筛选高意向客户给你跟进。

**售后支持**
使用问题、故障排查、退款处理。
第一时间安抚客户情绪。

Vigma社群里有位做SaaS的独立开发者，用AI客服处理了70%的咨询。
每天省下3小时。

## 三种实现方式

**方式一：ChatGPT + 知识库**

用OpenAI的API，接入你的产品文档。
适合：技术能力强的开发者
成本：按token计费，约0.002美元/千token

**方式二：第三方AI客服工具**

如：智齿客服、UDESK、美洽
提供可视化配置，无需代码
适合：非技术背景
成本：几百到几千元/月

**方式三：Notion AI / 飞书智能伙伴**

把常见问题整理成知识库。
用AI辅助回复，人工审核发送。
适合：咨询量不大的起步阶段
成本：几乎免费

## 实战：搭建简单的AI客服

以ChatGPT为例：

**第一步：准备知识库**

整理常见问题：
```
Q: 产品支持哪些支付方式？
A: 支持微信支付、支付宝、银行卡。

Q: 如何退款？
A: 购买7天内可申请退款，联系客服处理。

Q: 有免费试用吗？
A: 有，新用户可试用7天全部功能。
```

**第二步：写系统提示词**

```
你是[产品名]的客服助手。
你的职责是：
1. 友好地回答客户问题
2. 如果问题超出知识库范围，转接人工
3. 不要编造信息

知识库：
[粘贴上面的Q&A]

语气：亲切、专业、简洁
```

**第三步：接入聊天窗口**

用ChatGPT API + 简单的Web界面。
或者接入现有的客服系统。

## 关键：人机协作

AI客服不是万能的。

**适合AI处理：**
- 常见问题
- 简单咨询
- 非工作时间

**必须人工处理：**
- 复杂技术问题
- 投诉和负面情绪
- 大额订单谈判
- 特殊情况

设置转人工规则：
- 客户要求"找真人"
- AI连续3次无法回答
- 涉及退款/投诉

## 优化AI客服的5个技巧

**1. 持续更新知识库**

每出现一个新问题，就添加到知识库。
AI会越用越聪明。

**2. 收集客户反馈**

让客户评价AI回答是否有帮助。
根据反馈优化提示词。

**3. 个性化回复**

接入客户数据：
"您好张先生，看到您上周购买了XX产品..."

**4. 多语言支持**

AI自动识别语言，用客户母语回复。
一人公司也能服务全球客户。

**5. 主动服务**

不只是回答问题。
主动询问："还需要我帮您什么吗？"

## 常见问题

**Q: AI回复会不会很机械？**
取决于提示词。写得好，客户分辨不出来。

**Q: 客户会反感AI客服吗？**
只要解决问题快，客户不在乎是人是AI。

**Q: 需要技术背景吗？**
用第三方工具不需要。自建需要一点。
