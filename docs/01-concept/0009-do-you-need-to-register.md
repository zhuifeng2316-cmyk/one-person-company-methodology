# 一人公司需要注册吗

你刚决定做一人公司。

第一个问题不是"做什么"。
是"要不要注册公司"。

大部分人卡在这里三个月。
他们以为注册公司是门槛。
真相是：注册公司从来不是门槛，做出第一个付费客户才是。

## 先跑起来，再合法化

Vigma的"7天验证法"说得很清楚：

花7天时间，用最小成本验证你的产品想法是否有人愿意付费。

在这7天里，你不需要公司。
你需要的是一个收款方式。

微信、支付宝、个人银行卡。
足够了。

## 什么时候该注册

不是有收入了就要注册。
是收入稳定了才注册。

我的判断标准：
- 月收入超过当地免税额度
- 需要给客户开发票
- 有长期合作的B端客户
- 收入连续3个月稳定

在这之前，你只是个"有副业的个人"。
不是"公司"。

## 注册什么类型

在中国，一人公司通常三种选择：

**个体工商户**
适合：服务类、咨询类、小规模
优点：简单、税低
缺点：无限责任、难融资

**个人独资企业**
适合：有一定规模的个人业务
优点：比个体户正规
缺点：同样无限责任

**有限责任公司**
适合：有规模化打算、需要品牌背书
优点：有限责任、正规
缺点：成本高、手续多

## 一个误区

很多人以为注册公司=更专业。

错了。

专业度体现在你的交付质量。
不是营业执照上的几个字。

我见过太多人，公司注册好了，logo设计好了，网站上线了。
然后发现：没有客户。

顺序搞反了。

## 正确的顺序

1. 找到第一个愿意付费的人
2. 完成第一笔交易
3. 验证模式可行
4. 收入稳定后，再考虑注册

公司注册是结果，不是起点。

## 常见问题

**Q: 不注册公司能签合同吗？**
能。以个人名义签。或者用电子合同平台。

**Q: 客户要发票怎么办？**
去税务局代开。或者找财税公司挂靠。

**Q: 收入要交税吗？**
要。但个人劳务报酬的免税额度足够你起步。

**Q: 什么时候必须注册？**
当你需要"公司"这个身份去谈大客户、入驻平台、或者规模化扩张时。

别让注册公司成为你的拖延借口。

先做出价值。
再考虑形式。
