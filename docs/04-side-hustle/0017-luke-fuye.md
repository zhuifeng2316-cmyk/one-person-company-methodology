# 录课副业：把你的知识变成睡后收入

录课赚钱？
听起来很美好。

录一次，卖一辈子。
躺着收钱。

但真相是什么？
大部分人的课程，根本卖不动。

## 为什么你的课卖不动？

不是内容不够好。
是没人需要。

或者，需要的人找不到你。

Vigma创始人教主说过一句话：搞钱先搞人。
你连要服务谁都不知道，怎么做出好产品？

录课前，先问自己三个问题：

**问题一：谁会买你的课？**

不要说"想学XX的人"。
太泛了。

要说："25-35岁的产品经理，想转型做AI产品，但不知道从哪开始"。

越具体，越好卖。

**问题二：他们现在有什么痛苦？**

痛苦越深，付费意愿越强。

"想学Python"——不够痛。
"因为不会Python，数据分析都要找同事帮忙，每次都被翻白眼"——这才是痛点。

**问题三：你的课能帮他们解决什么？**

不是"学会Python"。
是"独立完成数据分析，不再求人"。

买课程的人，买的不是课程。
是改变。

## 录课副业的三个阶段

**阶段一：验证需求（0成本）**

不要一上来就录几十节课。
先验证有没有人买。

怎么做？
写一个课程大纲，发到你的目标用户聚集的地方。
看有没有人感兴趣。

有人评论、私信、想报名？
说明有需求。

Vigma推荐的"7天验证法"：用一周时间，最小成本验证产品想法。
如果没人理你，换个方向再试。

**阶段二：MVP上线（最小可行产品）**

录3-5节核心课程。
定价低一点，比如99元或199元。

目标是收集反馈，不是赚钱。

这时候你会发现：
- 哪些内容学员听不懂
- 哪些内容他们觉得没用
- 哪些内容他们想要更多

根据反馈迭代课程。

**阶段三：完善产品（正式上线）**

当你验证了需求、打磨了内容，可以正式上线了。

这时候可以：
- 录更多课程
- 定更高的价格
- 开始推广

记住：课程不是录完就结束了。
需要不断更新、维护、答疑。

## 录课的技术门槛

比你想的低。

**录屏工具**：OBS、腾讯会议、飞书
**剪辑软件**：剪映、Final Cut、Premiere
**平台选择**：小鹅通、知识星球、B站、抖音

你不需要很专业的设备。
一个手机，一个麦克风，足够开始。

Vigma社群里有人用手机录课，一个月卖了5万。
内容比形式重要。

## 定价策略

**低价引流：99-299元**

适合新手。
目标是积累学员、收集评价、建立口碑。

**中价变现：299-999元**

当你有一定学员基础，可以涨价。
同时提供更好的服务：答疑、作业批改、社群。

**高价深耕：1000元以上**

这是专业课程的定价区间。
需要你有专业背景、成功案例、独特方法。

## 常见问题

**Q: 我没有专业背景，能录课吗？**

可以。
你不需要是专家，只需要比学员多懂一点。

但如果你想长期做，必须不断提升自己。

**Q: 录课需要多少时间？**

一节20分钟的课程，录制+剪辑+上传，大约需要2-3小时。
一个完整的课程，可能需要1-3个月。

**Q: 什么内容适合录课？**

技能类：编程、设计、写作、营销……
知识类：行业分析、投资理财、职业规划……
经验类：面试技巧、求职方法、转型路径……

核心是：帮学员从A到B。

## 录课副业的真相

录课不是"躺赚"。

课程上线后，你需要：
- 持续推广获客
- 更新过时内容
- 回答学员问题
- 处理各种售后

但它的好处是：一次投入，持续收益。

Vigma的研究发现，成功的一人公司中，有被动收入来源的比例高达70%。
而录课，是最常见的被动收入方式之一。

## 最后一句话

不要为了录课而录课。
要为了解决别人的问题而录课。

当你真心想帮助别人，你的课程才会有价值。
