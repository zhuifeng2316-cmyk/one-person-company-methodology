# 咨询服务副业：把经验变成钱

你觉得自己的经验值多少钱？

很多人觉得自己"没什么特别的"。工作几年，没什么亮点。

但真相是：每个人都有可以卖的经验。

## 什么是咨询服务

咨询服务 = 出售你的专业知识和建议。

形式可以是一对一、语音、文字、会议...

定价从99到9999都可以。根据你的专业程度和客户付费能力决定。

## 谁可以做咨询服务

如果你在某个领域有超过1年的经验，你就可以做咨询。

比如：
- 帮新人改简历、模拟面试（职场咨询）
- 帮创业者梳理商业模式（创业咨询）
- 帮用户解决健康问题（营养咨询、健康管理）
- 帮家长解决育儿问题（育儿咨询）
- 帮程序员优化简历、准备面试（技术咨询）

Vigma社群里有句话："你不需要成为专家。你只需要比你的客户更懂一点点。"

## 怎么做

第一步：定位

不要什么都做。

选一个你能持续输出的方向。

公式：你能帮谁 + 解决什么问题 + 怎么解决

比如："帮程序员优化简历，拿到大厂offer"

第二步：准备产品

最简单的方式：1小时语音咨询。

定价：299-999元。

包含：1小时语音 + 事后发一份文字总结 + 7天内答疑

第三步：获客

在你已有的圈子里发："接简历优化/职业规划/XX咨询。"

不需要打广告。口碑介绍是最快的获客方式。

可以在知乎、公众号、小红书输出专业内容。内容会持续给你带来咨询客户。

第四步：交付

咨询前：让客户填一个表格，了解他的问题

咨询中：多听，多问，引导客户自己找到答案

咨询后：发一份详细的行动清单给客户

## 定价策略

刚开始可以低价（比如99元）积累案例。

做5-10个案例后，提价到299。

再做10个，提价到599。

慢慢涨。

你的经验越来越丰富，客户越来越信任你。

## 一个真实案例

Vigma社群里有位做技术面试咨询的成员。他原来是大厂程序员。

刚开始在知乎写面试相关回答。半年积累了几千粉丝。

然后推出"1小时模拟面试"服务，定价299元。

第一个月做了5单。第二个月做了12单。

现在他每月咨询收入2万+，还开了训练营。
