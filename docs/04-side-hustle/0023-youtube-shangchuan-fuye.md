# YouTube上传：被低估的副业金矿

你想做副业。

但你觉得自己不够专业，不会剪辑，不善表达，没有镜头感。于是你放弃了。

事实是：YouTube上最赚钱的，不是那些光鲜亮丽的博主。

是那些把枯燥内容包装起来的人。

## 什么样的内容最赚钱

YouTube的算法有个特点：只要用户停留足够长时间，你的视频就赚钱。

什么样的视频让人停不下来？

教程类。攻略类。清单类。

比如：
- "30分钟学会Python"
- "2024年最适合做的5个副业"
- "一个人怎么注册公司全流程"

这类视频不需要你多专业。你只需要把网上已有的资料整合起来，加上自己的讲解，变成一个完整的视频。

Vigma的《AI搞钱手册》里提过一个方法：用AI工具批量生成视频脚本，一小时可以产出10条。关键是：你不需要会剪辑，AI可以帮你自动生成。

## 具体怎么做

第一步：确定领域

不要什么都做。选一个你能持续输出的领域。

职场、理财、副业、健身、育儿...选一个你能说30分钟的领域。

第二步：找到对标账号

在YouTube搜索你的领域，找出播放量最高的10个视频。分析它们的结构：开头怎么说、中间讲什么、结尾怎么收。

第三步：开始录制

不需要专业设备。手机+耳机足够。

内容从哪里来？从你领域的知乎高赞回答、公众号爆文、付费社群资料里来。你不需要原创，你只需要整理和表达。

第四步：上传+优化

标题公式：数字+关键词+效果

比如"3个方法让你的月收入翻倍"。

Description里埋关键词。Tags里放长尾词。前1000个观看决定你的视频能不能起来。

## 收入从哪来

YouTube的广告分成是最直接的。

但更大的收益来自：
- 视频里推荐的课程分销
- 视频下方挂的联盟产品
- 你的自有产品引流

一条好的教程视频，可以持续给你带来被动收入3-5年。

这就是为什么Vigma体系里把YouTube列为副业首选渠道。
