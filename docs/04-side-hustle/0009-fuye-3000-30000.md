# 副业月入3000和月入3万的差距

同样的副业方向。

有人月入3000。有人月入3万。

差距是什么？

不是努力程度。不是时间投入。

是系统。

## 没有系统的人：卖时间

你做设计接单。100块一张图。

一个月接30单。月入3000。

然后你很累。质量下降。客户投诉。

你发现：你的收入等于单价乘以数量。

数量有上限。时间有上限。

这是线性收入。

也叫：卖时间。

## 有系统的人：卖杠杆

还是做设计。

他做了一套设计模板。上传到平台。

用户自助购买。自动发货。

一份模板卖了100份。100块一份。

月入1万。他可能只花了1周做这套模板。

剩下的时间干嘛？

再做一套。

这是指数收入。

也叫：卖系统。

## 三个能让你从3000到3万的杠杆

**1. 产品化**

把你的服务变成可复制的产品。

设计→设计模板。

咨询→录播课。

写作→电子书/专栏。

一次投入，多次变现。

**2. 自动化**

把销售流程自动化。

用平台、用工具、用流程。

不手动回复每一个咨询。

**3. 定价权**

从"按项目收费"到"按价值定价"。

你的价值是什么？

帮客户解决了什么问题？这个问题的价值是多少？

Vigma社群中有个做PDF工具的开发者，他的定价逻辑是："我的工具帮用户每月节省20小时，按100块/小时算，价值2000。"

他从39块的订阅费起步。现在月收入6位数。

## 从3000到3万，需要什么？

需要你停止只卖时间。

开始思考：我的能力怎么变成产品？怎么变成系统？

这不是一夜之间的事。

但它是唯一正确的方向。
