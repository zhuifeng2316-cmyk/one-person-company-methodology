# 月入4万的咨询顾问：他只服务一种人

Tom原来在大公司做战略咨询。

年薪50万。表面光鲜。

但他很累。每天通勤2小时。每周出差3天。见不到孩子。

所以他辞职了。

开始做一人咨询公司。

## 一个人怎么做咨询

Tom的做法很简单：

他只做一件事——帮SaaS创业公司优化定价策略。

不做什么？

不做品牌咨询。不做营销咨询。不做组织管理。

只做定价。

为什么？因为定价是他最擅长的事。也是SaaS公司最愿意花钱的事。

Vigma的"三圈定位法"说的就是这个道理：你擅长的 × 市场需要的 × 你喜欢的。三者交集，就是你的黄金定位。

Tom的交集很清晰：定价策略。

## 咨询顾问的核心资产

你以为咨询顾问卖的是时间？

不是。咨询顾问卖的是**认知差距**。

客户不知道怎么定价。你知道。这个"知道"，就是你的产品。

所以你的核心资产不是你的时间。是你的认知。

认知怎么积累？

Tom的方法：每个客户做完咨询，写一份案例总结。做了50个客户之后，他拥有了50个定价案例。

这些案例就是他的壁垒。新进来的竞争者没有这些经验。

Vigma社群里的顾问们也有类似的习惯。教主建议每次做完咨询都做复盘：什么方法有效、什么方法无效、客户真正关心的是什么。这些复盘积累下来，就是你的方法论。

## 一人咨询公司的定价策略

Tom的定价很有意思：

- 一次1小时咨询：免费（引流用）
- 定价诊断报告：5000元
- 定价优化项目：2万元
- 长期顾问：月费1万

从免费到付费，客户的心理门槛很低。

先让你免费体验我的能力。觉得有价值了，自然会买贵的。

## 怎么找到客户

Tom的客户来源只有两个：

**第一：内容**

他在知乎和小红书上写定价相关的内容。不是泛泛而谈的"定价很重要"。而是非常具体的"SaaS定价的3个常见错误"。

这类内容直接吸引目标客户。搜"SaaS定价"的人，大概率就是需要这个服务的人。

**第二：口碑**

每个满意客户给他介绍1-2个新客户。做了半年之后，他基本不需要主动找客户了。

Vigma的"搞钱先搞人"方法论在这里体现得淋漓尽致：你把当前客户服务好了，他们会帮你找到下一个客户。

## 常见问题

**Q: 一个人做咨询能赚多少？**

看你的定价和客户量。月入1-5万很常见。做得好的月入10万+。

**Q: 需要什么资质吗？**

不需要证书。客户买单是因为你解决问题，不是因为你有证。

**Q: 怎么防止客户跟别人做？**

签合同，设服务期限。但更重要的是：你的方法论是独特的，客户换人也要重新磨合。

你不是在卖时间。

你是在卖别人不知道的东西。

这就是咨询的本质。
