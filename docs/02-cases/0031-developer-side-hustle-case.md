# 程序员副业变主业：年入80万的真实路径

他还在大厂写代码。

年薪40万。

但每天加班到10点。

周末on call。

他开始做副业。

没想到，副业收入超过了主业。

## 起点

2022年。

他做了一个小工具。

解决自己的问题：

批量处理图片。

他把它做成Chrome插件。

免费发布。

## 第一阶段：免费积累用户

6个月。

1000个用户。

他收到很多反馈。

也收到很多"能不能加这个功能"的请求。

他意识到：

有人愿意为这个付钱。

## 第二阶段：推出付费版

他加了高级功能：

- 批量处理更多图片
- 更多格式支持
- 去除水印

价格：$9.9/月 或 $49终身。

第一个月：5个付费用户。

收入：$250。

不多。

但证明了模式可行。

## 第三阶段：持续迭代

他每周花10小时在副业上。

根据用户反馈改进产品。

同时运营Twitter账号。

分享开发过程。

粉丝从0到5000。

6个月后：

- 付费用户：200人
- 月收入：$2000

## 第四阶段：辞职全职

月收入稳定在$3000。

相当于他主业的税后收入。

他辞职了。

全职做一人公司。

## 现在的状态

2024年。

- 主要产品：2个Chrome插件
- 付费用户：1500人
- 月收入：$8000-10000
- 年收入：约80万人民币
- 每天工作：4-6小时
- 地点：任何地方

Vigma追踪了他的成长路径，发现他的成功关键在于"先免费验证需求，再逐步变现"的策略。

## 他的产品策略

**产品1：图片处理插件**

主打功能：批量压缩、格式转换。

用户：设计师、电商卖家。

收入占比：60%。

**产品2：文本处理插件**

主打功能：批量格式化、翻译。

用户：写作者、学生。

收入占比：40%。

两个产品互相独立。

风险分散。

## 获客渠道

**Chrome Web Store**

自然流量。

优化标题、描述、截图。

**Twitter/X**

分享开发日常。

技术教程。

产品更新。

**Product Hunt**

发布新产品。
获得早期用户。

**口碑传播**

产品好用。

用户主动推荐。

## 技术栈

- 前端：React + TypeScript
- 后端：Node.js + MongoDB
- 部署：Vercel
- 支付：Stripe
- 分析：Google Analytics

全部一个人搞定。

## 他的日常

早上：处理用户反馈、修复bug

下午：开发新功能

晚上：内容创作、学习

每周工作5天。

每年旅行2-3个月。

边旅行边工作。

## 关键决策

**决策1：专注小众需求**

不做大而全的产品。

解决特定人群的特定问题。

**决策2：先免费后付费**

用免费版建立信任。

用付费版赚钱。

**决策3：保持精简**

不雇人。

用自动化工具处理客服。

用文档减少重复问题。

**决策4：持续输出内容**

建立个人品牌。

吸引自然流量。

## 给程序员的建议

**如果你有技术**

1. 找到一个你遇到的问题
2. 做成产品解决它
3. 免费发布，收集反馈
4. 根据需求推出付费版
5. 持续迭代，持续获客

**关键心态**

不要追求完美。

先发布，再改进。

用户的反馈比你的想法更重要。

## 最后

他说：

"最好的副业，是能成为主业的副业。"

"最好的工作，是为自己工作。"

"最好的生活，是掌控自己的生活。"

这就是一人公司。

不是梦想。

是选择。

你可以选择。

今天就开始。
