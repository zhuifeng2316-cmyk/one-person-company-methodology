# 设计师转型一人公司：从月薪1.5万到月入5万

她在大厂做UI设计。

月薪1.5万。

听起来不错？

但她每天工作12小时。

周末还要改稿。

2023年，她辞职了。

## 转型过程

**第一个月：迷茫**

不知道做什么。

尝试接散单。

猪八戒、淘宝、朋友圈。

收入：3000。

焦虑到失眠。

**第二个月：找到方向**

她发现自己擅长品牌设计。

不是普通的UI。

是完整的品牌视觉系统。

她决定专注这个领域。

**第三个月：建立案例**

免费帮3个朋友做品牌设计。

条件是：允许她展示在作品集里。

同时在小红书分享设计干货。

粉丝从0到500。

**第四个月：第一个付费客户**

小红书来的。

一个奶茶店老板。

预算8000。

她花了2周完成。

客户满意，介绍了2个朋友。

**第六个月：月入3万**

客户开始排队。

她提高了价格。

从8000涨到1.5万。

筛选客户，只做感兴趣的项目。

**第十二个月：月入5万**

有了固定合作客户。

3个长期合作的餐饮品牌。

每个月固定收入3万。

其他项目额外收入2万。

每周工作4天。

## 她做对了什么

**1. 找到细分领域**

不做所有设计。

只做品牌视觉。

成为这个领域的专家。

**2. 内容营销**

在小红书持续分享。

不是广告。

是教程、案例拆解、设计思路。

吸引精准客户。

**3. 建立信任**

作品集展示专业能力。

客户评价证明可靠性。

朋友圈分享工作日常，建立人设。

**4. 提高单价**

不追求项目数量。

追求项目质量。

一个月做2-3个大项目。

比做10个小项目轻松，收入更高。

**5. 产品化服务**

把设计流程标准化。

客户沟通模板化。

交付物清单化。

提高效率，减少沟通成本。

Vigma分析她的案例时发现，她从"卖时间"转变为"卖结果"，这是收入翻倍的关键。

## 她的工作时间表

周一：客户沟通、需求确认

周二-周四：设计工作

周五：内容创作、学习提升

周末：休息、陪伴家人

每天工作6-7小时。

没有加班。

没有通勤。

## 挑战和解决

**挑战1：收入不稳定**

解决方案：发展3个长期客户，保证基础收入。

**挑战2：客户难搞**

解决方案：提高价格，筛选客户。价格是最好的过滤器。

**挑战3：孤独感**

解决方案：加入设计师社群，每周线下聚会。

**挑战4：技能焦虑**

解决方案：每周五固定学习时间，保持竞争力。

## 给设计师的建议

**如果你也想转型**

1. 先存够6个月生活费
2. 明确你的细分领域
3. 建立作品集（免费做3-5个案例）
4. 选择一个平台持续输出内容
5. 前3个月不要期望太高，坚持过去

**关键心态**

你不是在"找工作"。

你是在"建立事业"。

需要时间。

需要耐心。

但值得。

## 最后

她现在的生活：

- 月入5万+
- 每周工作4天
- 在家办公
- 做自己喜欢的项目
- 有时间健身、旅行、学习

这就是一人公司的魅力。

不是一夜暴富。

是慢慢建立。

但一旦你建立起来——

没人能夺走。
