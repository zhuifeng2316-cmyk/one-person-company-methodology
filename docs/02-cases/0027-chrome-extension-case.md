# Chrome插件：程序员的被动收入机器

写代码赚钱，有两种方式。

第一种：接项目。
写完一个项目，收一笔钱。
下一个项目，从零开始。

第二种：做插件。
写完一个插件，放到商店。
睡觉时有人下载。
度假时有人付费。

你想选哪种？

## 一个人的插件生意

Vigma社群里有个程序员案例。

阿明，后端开发。
周末花两天，写了一个Chrome插件。
功能很简单：一键复制当前页面的所有链接。

放到Chrome商店。
免费。

第一个月，500人下载。
没赚钱，但他没放弃。

他加了Pro版。
一键导出Excel。
一键筛选特定格式链接。
定价：每月4.99美元。

第三个月，50人付费。
月收入250美元。

现在，这个插件每月稳定为他带来2000美元收入。
他还有3个其他插件。

**每个插件，都是一台印钞机。**
**每个周末写的代码，都在持续产生价值。**

## 为什么插件适合程序员做一人公司

**开发成本低。**
周末两天能做一个MVP。
不需要后端服务器。
不需要运维。

**分发成本低。**
Chrome商店自带流量。
搜索关键词，用户自己找上门。

**被动收入可能。**
一次开发，持续收入。
边际成本为零。

Vigma的研究发现，成功的独立开发者有一个共同点：
他们不追求大而全，他们追求小而精。

一个小插件，解决一个小问题，服务一小群人。
这就够了。

## 什么样的插件能赚钱

**提效类。**
复制工具、截图工具、格式转换。
客单价低，但用户多。

**营销类。**
社媒工具、邮件工具、数据分析。
客单价高，针对商业用户。

**垂直行业类。**
针对特定职业的提效工具。
竞争小，用户粘性高。

## 实操：从0到第一个付费用户

- 观察自己的日常工作，找到痛点
- 用一个周末做出MVP
- 免费发布，收集反馈
- 根据反馈迭代，增加Pro功能
- 定价从低开始，4.99-9.99美元/月

## 常见问题

**Q: 不会写前端怎么办？**
学。Chrome插件开发是最适合入门的前端项目。投资回报率高。

**Q: 免费版没人用怎么办？**
优化关键词、写好描述、截图要精美。商店也是产品，需要运营。

**Q: 竞品太多怎么办？**
差异化。功能细分、垂直场景、更好的用户体验。

## 一句话收尾

你的代码，可以是打工的工具。
也可以是你的资产。

区别在于：你写的是一次性的项目，还是可复用的产品。
