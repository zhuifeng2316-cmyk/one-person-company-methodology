# 模板售卖：做一次，卖一万次

做一次工作，只赚一次钱。
这是打工。

做一次工作，卖一万次。
这是一人公司。

模板售卖，就是把你的时间变成资产。

## 一个人的模板生意

教主在Vigma社群分享过一个案例。

小美，设计师。
以前接单做PPT，一份500。
熬夜做图，客户改来改去。
累死累活，一个月赚8000。

后来她换了个思路。

她把自己的PPT模板放到网上卖。
一套39元。

第一个月，卖了200份。
收入7800。
跟以前一样多。
但她只花了3天做模板。

第二个月，卖了500份。
收入19500。
她什么都没做。
模板在自动卖。

现在，她有30套模板。
每月被动收入稳定在3万以上。

**她不再熬夜改稿。**
**她不再面对挑剔客户。**
**她只做一件事：做出好模板。**

**一次生产，无限销售。**
边际成本为零。
睡觉时，模板在卖。
度假时，模板在卖。

**不需要客户沟通。**
没有改稿。
没有催款。
没有情绪劳动。

**可以从小做起。**
先做一套，看看市场反应。
不需要大投入。

把服务变成产品。
把时间变成资产。

## 哪些模板能卖钱

**Notion模板。**
项目管理、习惯追踪、知识库。
Notion用户增长快，需求大。

**PPT模板。**
商务汇报、工作汇报、路演。
每份39-99元，走量。

**Excel模板。**
财务报表、项目管理、数据分析。
专业度高，定价可以更高。

**设计模板。**
简历模板、社媒模板、海报模板。
设计师的优势赛道。

## 实操：如何开始

- 找到你的技能所在领域
- 分析同类模板，找到差异化点
- 做3-5套精品模板（不是多，是精）
- 上传到平台（Notion、Gumroad、闲鱼、小红书）
- 持续优化，根据销量反馈迭代

## 常见问题

**Q: 定价多少合适？**
低价走量：19-49元。高价精品：99-299元。看你定位。

**Q: 哪个平台好？**
国内：闲鱼、小红书、知识星球。国外：Gumroad、Notion Marketplace、Etsy。

**Q: 没有设计功底怎么办？**
Notion模板不需要设计功底。内容质量更重要。

## 一句话收尾

你的技能，可以变成服务，服务一个客户。
也可以变成模板，服务一万个客户。

选择权在你。
