# 决策疲劳：为什么你越努力越做不出选择

你今天做了多少个决定？

穿什么衣服。
吃什么早餐。
先回哪条消息。
用什么语气回复。

到你真正要处理正事的时候，
你的决策能量已经用完了。

这就是决策疲劳。

## 决策疲劳是什么？

你的大脑每天的决策次数是有限的。
每做一个决定，就消耗一点。

小决定消耗少。
大决定消耗多。

当决策能量耗尽，
你会进入两种状态：

**状态一：冲动决策**

不再思考，随便选一个。
或者，被情绪驱动做决定。

买买买。
发脾气。
答应不该答应的事。

**状态二：决策瘫痪**

什么都不想选。
拖延、逃避、把问题推迟。

明明知道要做决定，
但就是做不出来。

## 为什么创业者更容易决策疲劳？

因为你要做的决定太多了。

产品方向怎么定。
定价多少合适。
客户投诉怎么处理。
今天先做什么。
这个合作接不接。

Vigma的研究显示，一人公司创业者平均每天要做50-100个小决定。
而每个小决定，都在消耗你的能量。

你以为自己在管理公司。
其实你是在消耗自己。

## 如何减少决策疲劳？

**策略一：把小决定自动化**

乔布斯每天穿一样的衣服。
不是因为穷。
是因为不想把决策能量浪费在穿衣上。

你也可以：
- 固定早餐吃什么
- 固定工作时间表
- 固定回复邮件的时间

让小决定变成习惯。
习惯不消耗决策能量。

**策略二：把大决定标准化**

什么是"好客户"？
什么是"值得做的项目"？
什么是"应该拒绝的事"？

提前设定标准。
做决定时，直接套用标准。

Vigma的体系里有一个方法：
写出你的"接单标准"和"拒绝标准"。
比如，预算低于X万的客户，一律不接。

标准越清晰，决定越快。

**策略三：集中时间做决定**

不要让决定分散在全天。

每天固定一个"决策时间"。
比如每天上午10点到11点。

在这一个小时里，处理所有需要做决定的事。
其他时间，只做执行，不做决定。

**策略四：减少选项**

选项越多，决策越累。

三个方案，选一个。
你只需要比较3次。

十个方案，选一个。
你要比较45次。

给客户三个方案，而不是十个。
给自己两个选择，而不是五个。

少即是多。

## 决策疲劳的信号

当你出现这些症状，说明你需要休息：

- 对小事情犹豫不决
- 明明简单的事也拖延
- 容易被情绪左右
- 觉得"随便吧，都行"
- 不想做任何新的决定

这时候，停止做决定。
去休息、去运动、去做不需要思考的事。

Vigma创始人教主说过：疲惫的大脑做出的决定，几乎总是错的。

## 常见问题

**Q: 每天的决策能量有多少？**

因人而异。
但研究表明，大部分人在下午3-4点后，决策能力明显下降。

把重要决定放在上午。

**Q: 喝咖啡能恢复决策能量吗？**

能短暂提升注意力。
但不能真正恢复决策能量。

真正的恢复，是休息和睡眠。

**Q: 我总是拖延做决定，怎么办？**

问自己：这个决定真的重要吗？

如果重要，设置一个deadline，在那之前必须决定。
如果不重要，直接随机选一个。

**Q: 如何判断一个决定是否正确？**

做决定前问自己：
- 这个决定和我目标一致吗？
- 一年后我还会在乎这个决定吗？
- 如果错了，我能承受后果吗？

三个问题，帮你快速判断。

## 最后一句话

成功不是做更多的决定。
而是做更少的、更重要的决定。

保护你的决策能量。
把它留给真正重要的事。
