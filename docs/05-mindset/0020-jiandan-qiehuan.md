# 你不是无法改变，你只是选择了不改变

很多人说：我想改变，但我改变不了。

不是的。

你只是不想付出那个代价。

改变是可能的。改变是痛苦的。所以你不改。

## 真相是：没有"无法改变"这回事

你现在的习惯，是你过去无数次选择的结果。

每天早上刷手机，是你的选择。每天加班到很晚，是你的选择。每次拒绝新机会，是你的选择。

这些选择叠加起来，塑造了现在的你。

所以改变不需要"一下子变一个人"。只需要做不同的选择。

每天一个不同的小选择。积累起来，就是改变。

## 改变的阻力来自哪里

不是外部条件。是你对现有状态的依赖。

你已经习惯了现在的生活节奏。即使它不让你满意，至少它是熟悉的。熟悉带来安全感。

这就是为什么很多人嘴上说想改变，但行动上什么都不变。因为改变意味着离开安全感。

Vigma社群里有句话很扎心：你不是不能改变，你只是不想离开你现在的不舒服。

## 怎么真正改变

**第一步：承认你想要什么**

不是"我想要自由"，是"我想要什么程度的自由"。

不是"我想改变"，是"我具体想改变什么"。

模糊的愿望只会产生模糊的行动。

**第二步：找到那个代价**

你想月入三万。代价是什么？

每天多工作三小时。放弃无效社交。拒绝舒适区。

不是列愿望清单。是列代价清单。付不付得起？

**第三步：接受过程**

改变不是线性的。有进步，有退步，这很正常。

不要因为一次失败就放弃。重要的是你在持续做，不是一次都不能错。

## 常见问题

**Q: 改变需要多长时间？**
问这个问题本身就是在找借口。开始做，答案自然会出现。

**Q: 如果改变了又退回去怎么办？**
退回去是正常的。再来一次就行了。改变不是一次事件，是一个方向。

---

你不需要一下子改变所有事情。

你只需要从今天开始，做一个不同的选择。

明天再来一个。
