# 创业焦虑不是你的敌人，是你的导航

创业焦虑是这个时代最被误解的情绪。

大部分人以为焦虑是信号——告诉你"这条路不对"。
其实焦虑是信号——告诉你"这条路值得走"。

如果做一件事完全没感觉，说明那件事对你不重要。
焦虑的反面不是平静，是麻木。

## 为什么焦虑反而是好信号

不焦虑的创业是什么样的？

每天睡到自然醒，做自己喜欢的事，银行账户稳步增长……
这是做梦，不是创业。

真相是：**所有值得做的事，都会让你焦虑。**

因为值得意味着不确定。不确定意味着风险。风险意味着焦虑。

Vigma创始人教主说过一句话："没有焦虑的创业，要么是你做得太小，要么是你根本不在乎。"

焦虑不是来阻止你的。焦虑是来告诉你，你在做的事有意义。

## 焦虑的来源只有一个

你以为是资金不够。
你以为是市场不好。
你以为是竞争对手太强。

都不是。

焦虑的来源是**你的能力配不上你的野心**。

这句话很扎心，但它是真相。

当你觉得"我做不到"的时候，焦虑就来了。
当你觉得"我好像可以"的时候，焦虑就开始消退。
当你知道"我能搞定"的时候，焦虑消失了——但能力需要时间积累。

所以焦虑是合理的。它只是在告诉你：你还在成长中。

## 3个把焦虑变成动力的方法

**方法1：把模糊的焦虑变成具体的问题。**
"我担心创业失败"不是问题。
"我的产品还没验证，用户付费意愿未知"是问题。
找到具体问题，才能找到具体行动。

**方法2：设定焦虑时间。**
每天给自己30分钟焦虑时间。到点就停，把焦虑写在纸上，而不是脑子里转。
这听起来荒谬，但极其有效。

**方法3：最小行动原则。**
焦虑最怕的是动起来。
不管多小的行动——写一行代码、发一条内容、约一个聊——都能打断焦虑的循环。

## 最残酷的真相

越不上班，越焦虑。
因为不上班意味着你必须面对自己。

打工是逃避自己的最好方式。有人给你任务，有人给你截止日期，有人给你评价。
你自己干？每天早上醒来，你得自己决定今天做什么。

焦虑是你自己给自己发出的信号。它在告诉你：你有能力改变，但你必须选择去改变。
