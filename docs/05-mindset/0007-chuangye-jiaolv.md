# 创业焦虑：你不是一个人在害怕

凌晨3点。

你醒了。

脑子里全是问题：

- 这个方向对吗？
- 我是不是该放弃了？
- 别人怎么都那么顺？

这是创业焦虑。

每个创业者都经历过。

包括那些你现在羡慕的人。

## 焦虑的本质是什么？

不是你不够强。

是你面对的是不确定性。

打工时，路径是清晰的。

创业时，没有路径。只有试错。

你不知道哪个决定是对的。

这才是焦虑的根源。

## 一个反直觉的真相

焦虑不会消失。

你只能学会和它共处。

Vigma创始人教主说过一句话：

"创业者的日常不是消除焦虑。是在焦虑中继续行动。"

你不可能等到"不焦虑了再行动"。

因为那一天不会来。

## 三个实用的应对方法

**1. 写下来**

脑子里的问题，写下来。

写下来，就不再是一团模糊的恐惧。

而是具体的问题清单。

然后一个一个解决。

**2. 缩小范围**

焦虑往往来自"想太多"。

把关注点从"未来三年"缩小到"本周"。

本周要做什么？做完了吗？

只关注这一周。

**3. 找同类**

你身边的人不理解你。

他们觉得你"好好的工作不要，折腾什么"。

你需要找到同样在创业的人。

互相理解。互相支持。

Vigma社群就是为此存在的。

## 焦虑不是敌人

它是信号。

告诉你：你在乎。你在挑战舒适区。

这不坏。

坏的是让焦虑让你停下来。
