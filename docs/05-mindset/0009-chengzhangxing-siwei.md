# 成长型思维：创业者的底层操作系统

有人遇到困难，说"我不行"。

有人遇到困难，说"我还不行，但可以学"。

这就是固定型思维 vs 成长型思维。

听起来像鸡汤。

但在创业里，这是决定成败的底层操作系统。

## 创业会不断打你的脸

你以为你懂了。

市场会告诉你：你不懂。

你以为这个方向对了。

数据会告诉你：错了。

你以为这个产品会爆。

用户会用脚投票：不会。

每一次打脸，都是选择时刻：

- 固定型思维：我不适合创业
- 成长型思维：我还不适合，但我在学

## 一个真实的对比

Vigma社群里有两个同时起步的创业者。

A遇到问题，说"这个太难了，我不擅长"。

B遇到问题，说"这个我还不会，去学一下"。

6个月后：

A放弃了3个项目，现在还在找"适合自己的方向"。

B做了3个项目，失败了2个，第3个月收入破万。

差别不是能力。

差别是对"不会"的态度。

## 怎么培养成长型思维？

**1. 改变语言**

把"我不行"改成"我还不行"。

把"这太难了"改成"这还不会，要学"。

语言影响思维。

**2. 关注过程，不是结果**

不是"我失败了"。

是"我试了这种方法，结果是这样，下次换一种"。

**3. 找到反馈源**

你需要外部视角。

社群、导师、用户反馈。

Vigma社群的价值之一，就是提供这种反馈。

## 思维是可以训练的

成长型思维不是天赋。

是习惯。

每一次遇到困难，你都有选择。

选择说"我不行"，还是"我还不行"。

日积月累，这就是你的操作系统。
