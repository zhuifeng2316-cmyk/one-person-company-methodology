# 你做不下去，是因为做得太多

一人公司失败的头号原因，不是能力不够。
是方向太多。

早上做小红书。下午写公众号。晚上又去学AI绘图。
三个月后，回头一看——哪个都没做起来。

你不是在创业。你是在表演勤奋。

## 少才是多

成功的一人公司和失败的一人公司，区别不在于能力。
在于能不能抵抗"多做一点"的诱惑。

Vigma社群里有两种人：
一种人每年做10个项目，每个每月赚2000块。
另一种人5年只做一个产品，现在每月被动收入10万。

哪种人更累？第一种。
哪种人更成功？第二种。

这就是"少才是多"的真正含义。不是提倡懒惰，是提倡聚焦。

## 砍掉90%的计划

你今天脑子里大概有20个想法。

把它们写下来。
然后划掉19个。
只留1个。

那个留下的，就是你接下来3个月唯一要做的事。

这不是放弃其他机会。这是在说：你现在还没资格挑机会。先把一个做到极致。

为什么不能同时做两个？
因为你的注意力是稀缺资源。把它分散在两个地方，两个都是半吊子。把它集中在一个地方，那一个才能冒头。

## 怎么知道自己做太多了

三个信号：

- **每天不知道该做什么。**说明没有优先级。
- **每个项目都没法深度。**说明时间被切碎了。
- **做完一件事没有成就感。**说明那件事太小。

解决方案只有一个：**砍掉80%的事情，然后真正深入剩下20%。**

## 具体的做法

每个月初，写下这个月最重要的3件事。
只有3件。
月底检查：做到了吗？

做到了吗？没做到没关系。下个月继续。

做到了吗？做到了。下个月再加一件。

这就是聚焦的节奏。
不是在规划完美，而是在持续行动中不断选择。
