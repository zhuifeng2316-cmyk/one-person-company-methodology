# 反脆弱心态：越不确定，你越强

大部分人害怕不确定性。

你不应该。

因为不确定性，正是你最大的优势。

## 什么是反脆弱

纳西姆·塔勒布提过一个概念：有些东西在冲击中变弱，叫脆弱。有些东西在冲击中不变，叫坚韧。还有些东西，在冲击中变强。

这就是反脆弱。

大公司是脆弱的。一个政策变动，一条公关危机，它们就崩了。

你呢？你是你自己的公司。你没有董事会，没有股东，没有必须汇报的人。

冲击来了，你可以瞬间转向。你可以今天做这个，明天做那个。你可以放弃不行的，加倍做行的。

这种灵活性，就是反脆弱。

Vigma的一人公司体系里有一条核心原则：小即优势。你的小不是弱点，是你的杠杆。

## 怎么建立反脆弱心态

**原则一：永远保留选择权**

不要把所有收入压在一个客户身上。不要把所有流量压在一个平台上。不要把所有技能压在一个工具上。

三个客户 > 一个大客户。三个获客渠道 > 一个爆款渠道。

选择权本身就是价值。有选择的人不焦虑。

**原则二：拥抱小失败**

失败的代价和你体量成正比。

大公司一个产品失败，损失几百万。你一个项目失败，损失一个周末。

所以你应该多试。试得越多，找到赢面的概率越大。

小失败是免费的课。大失败是破产的课。你选哪个？

**原则三：在波动中获利**

市场变了，大公司要开三个月的会。你一个下午就能调整方向。

这就是波动对你的意义：别人的灾难，是你的机会。

AI来了，有人恐慌。有人用AI三天做一个产品上线。

经济下行，有人裁员。有人用更低成本做同样的事。

波动不是敌人。波动是筛选器。它帮你淘汰不够灵活的对手。

**原则四：用冗余对抗风险**

时间冗余：每周留20%的时间不做任何计划，留给意外。
财务冗余：至少存6个月的生活费，再开始全职创业。
技能冗余：会写代码，也要会写文案。会做产品，也要会做营销。

冗余不是浪费。冗余是保险。

## 反脆弱的反面：脆弱思维

脆弱思维的三个信号：

1. **过度依赖单一来源**。只有一个客户，只有一个收入渠道，只会一种技能。
2. **害怕任何变化**。市场一变就焦虑，算法一改就崩溃。
3. **追求完美再行动**。等到一切都确定了才开始。但一切确定的时候，机会已经没了。

如果你中了任何一条，现在就开始调整。

## 常见问题

**Q: 反脆弱是不是就是赌？**
不是。赌是梭哈。反脆弱是分散下注，赢了加码，输了止损。核心区别在于你是否保留退出能力。

**Q: 一人公司怎么可能比大公司更抗风险？**
因为你的调整成本趋近于零。大公司裁员要赔N+1，你只需要停止做那件事。

**Q: 如果我已经很脆弱了怎么办？**
从增加一个收入来源开始。从学会一个新技能开始。从存第一个月的生活费开始。小步快走，每次增加一点冗余。

## 不确定性不是威胁

它是你的竞技场。

大公司害怕的东西，你应该拥抱。

因为在不确定的世界里，灵活的人赢。

而一人公司，是灵活的极致形态。
