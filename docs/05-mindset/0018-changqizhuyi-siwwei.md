# 你3个月后想成为什么样的人

你想过没有，3个月后的自己，应该是什么样子？

不是"我想赚更多钱"这种。是具体的、可视化的画面。

比如：3个月后，我能独立做一个小程序；6个月后，我每月副业收入5000；1年后，我有自己的付费社群。

这就是长期主义。

## 普通人为什么很难坚持

因为他们用短期视角看长期目标。

"我都发了一周公众号了，为什么还没人关注？"
"我都学了一个月Python了，为什么还找不到工作？"
"我都写了3篇文章了，为什么还没变现？"

真相是：3个月只是起步价。

Vigma创始人教主说过："大多数人高估了1年内能做的事，低估了10年内能做的事。"

## 怎么做到长期主义

方法1：设置里程碑

不要想"我要做一年"。想"这周我要完成什么"。

把一年拆成52周。每周一个里程碑。52周后，你回头看，会发现自己走了很远。

方法2：记录进步

每天写3行日记：
- 今天完成了什么
- 今天学到了什么
- 明天要做什么

这个习惯坚持一年，你会有惊人的变化。

方法3：远离噪音

短期成功的人会给你制造焦虑。他们告诉你"7天变现""月入10万"。

但你不需要关注他们。你只需要关注自己的节奏。

## 真正的自律是长期习惯

很多人以为自律是每天5点起床，每天跑10公里。

不是的。

真正的自律是：每天做一点，持续做很久。

每周写一篇文章。每月学一个新技能。每年尝试一个新项目。

这就是复利。
