# 你不是缺乏意志力。你是缺乏系统。

很多人问我：

"我白天上班，晚上想做副业，但回家就只想刷手机，怎么办？"

"我每次计划得很好，但坚持不了三天，怎么办？"

"我下载了十几个学习软件，但每个都只用了两天，怎么办？"

他们的共同假设是：
我意志力不够强。
我需要更努力。
我需要更自律。

这个假设，是错的。

## 意志力是一个谎言

你见过有人靠意志力戒烟成功吗？
有。
成功率多少？
3%到5%。

剩下95%的人呢？
他们用意志力戒烟，失败了。
然后他们责备自己意志力不够。

这不是意志力的问题。
这是系统的问题。

一个人戒烟成功，不是因为他意志力强。
是因为他改变了自己的系统。

他换了一个环境，身边没有人抽烟。
他找到了替代行为，想抽烟的时候嚼口香糖。
他把戒烟的动机写下来，放在每天能看到的地方。

这是一个系统。
不是意志力。

## 你的问题不是懒

你回家只想刷手机。
这不是因为你懒。
是因为你一天消耗了太多意志力。

白天开会、处理冲突、做不想做的事情。
意志力是消耗品。
消耗完了，就没了。

所以回到家，你本能地选择最省力的事情：
刷手机。

这不是失败。
这是人的正常生理机制。

## 怎么设计你的系统

**第一：提前设计你的行动**

不要在意志力最低的时候做决定。

早上起床后，第一件事是副业相关的事情。
因为这时候意志力最满。

不要等到晚上。
晚上留给执行，不留给决策。

**第二：把环境设计成你的盟友**

想晚上看书？
把书放在枕头边。
想早上写作？
前一天晚上把电脑打开。

你想做什么，就让这件事变得最容易。

反过来：
想戒掉刷手机？
回家第一件事，把手机放进抽屉。

环境设计，比意志力有效100倍。

**第三：从小到荒谬的承诺开始**

不要计划"每天写作2小时"。
计划"每天写50个字"。

这个承诺，小到不可能失败。
成功了，你可能会写更多。
失败了，你也不会有挫败感。

Vigma社群有一个"两分钟规则"：
任何一个任务，先做两分钟。
两分钟后，你可以停止。
大多数时候，你会继续做下去。

**第四：建立一个反馈循环**

没有反馈，就无法坚持。

今天做了什么事？
有什么结果？
哪里可以改进？

写下来。
每周看一次。

你会看到自己的进步。
这比任何外部激励都有效。
