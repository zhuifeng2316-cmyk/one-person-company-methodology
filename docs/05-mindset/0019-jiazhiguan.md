# 价值观不是用来装的，是用来做决定的

大部分人谈价值观，是因为别人的价值观听起来不错。

不是真的相信。

这就是为什么当压力来临，他们的"价值观"瞬间消失。

## 真正的价值观只在关键时刻出现

你遇到一个短期赚快钱的机会，但它违背了你的长期方向。你怎么选？

你有一个客户愿意付很多钱，但你不认同他的做法。你接不接？

这些问题没有标准答案。但你的答案，定义了你是什么样的人。

Vigma创始人教主说过一句话：搞钱先搞人。搞人的意思是，你得先知道自己是谁，才知道什么钱该赚，什么钱不该赚。

没有价值观，你只是在随波逐流。

## 价值观不是起点，是结果

很多人觉得：我要先有一套价值观，然后按价值观生活。

顺序错了。

价值观是行为的结果。不是你先有一套理论，而是你一直在做某些事情，慢慢总结出来你相信什么。

你不需要现在就有完整的价值观体系。你需要开始做选择，然后观察自己的反应。

每次选择，都是在定义自己。

## 怎么建立你的价值观

**方法一：记录你的"不"**

写下最近你拒绝的三件事。为什么拒绝？

不是因为"不划算"。是因为什么？

这个列表就是你的价值观雏形。

**方法二：观察你的愤怒**

什么让你生气？

别人做了什么事，你会愤怒？

愤怒背后是你在乎的东西。那就是你的价值观。

**方法三：想象你的葬礼**

不是 morbid，是工具。

想象你的葬礼。你希望人们怎么评价你？

那个评价，就是你想成为的人。然后倒推：你要做什么，才能成为那样的人。

## 常见问题

**Q: 价值观会不会限制赚钱？**
短期可能。长期不会。因为没有价值观的人，赚再多也空虚。而且没有价值观的生意，做不大。

**Q: 价值观可以改变吗？**
可以。但不是因为外部压力改变。是因为你自己成长了，主动选择改变。

---

价值观不是固定不变的东西。它是你每天选择的总和。

你选择做什么，不做什么，这就是你。
