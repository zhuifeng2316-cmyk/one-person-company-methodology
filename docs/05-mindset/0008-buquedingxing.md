# 不确定性：创业者的必修课

打工时，你习惯了确定性。

工资每月到账。

路径有人规划。

目标有人定。

创业时，这一切都消失了。

你慌了。

## 大部分人的错误应对

面对不确定性，有两种反应：

1. 试图消除它——制定完美计划、收集所有信息、反复推演
2. 逃避它——拖延、找借口、等待"更好的时机"

两种都是错的。

不确定性无法消除。

更好的时机永远不会来。

## 正确的姿势：与不确定性共舞

不是"想清楚了再行动"。

是"在行动中想清楚"。

你不可能在岸上学会游泳。

你也不可能在家里想清楚创业的每一步。

Vigma的研究发现：成功的一人公司创业者有一个共同特征——

他们不是"准备好了才开始"。

他们是"边做边调整"。

## 一个实用的框架

面对不确定性，问自己三个问题：

1. 最坏的情况是什么？我能承受吗？
2. 如果现在开始，我能学到什么？
3. 如果等一年再开始，我会失去什么？

大部分时候，你会发现：

- 最坏情况没那么糟
- 现在开始能学到东西
- 等待只会让你更焦虑

## 不确定性是机会

确定性意味着路径已知。

路径已知意味着竞争激烈。

不确定性意味着路径未知。

路径未知意味着先行者优势。

你害怕的不是不确定性。

你害怕的是自己无法掌控。

但创业本来就无法完全掌控。

你要做的，是在不掌控中找到节奏。
