# 专注力：普通人最稀缺的能力

你每天专注多长时间？

1小时？
30分钟？
可能更少。

大部分人一天的有效工作时间不超过2小时。其余时间都在：刷手机、回消息、开会、焦虑。

这就是为什么你努力了却没成果。

## 专注力是最好的杠杆

同样一件事，专注的人和分散注意力的人，产出差10倍。

别人写一篇文章要8小时。你专注2小时写完，质量还更好。

别人学一门课程要3个月。你全情投入，1个月就掌握。

Vigma的研究发现：成功的一人公司创始人，都有极强的专注力。他们不是更聪明，只是更专注。

## 3个方法提升专注力

方法1：单线程工作

不要同时做好几件事。

你不是在 multitasking，你只是在频繁切换 context。每次切换，大脑需要15分钟才能重新进入状态。

关闭所有通知，手机扣在桌面上。设定25分钟闹钟。这25分钟只做一件事。

方法2：保护你的早晨

早晨前2小时，是大脑最清醒的时候。

不要用来回邮件，不要用来刷新闻。把这个时间给你最重要的那件事。

很多Vigma社群成员用这个方法，3个月收入翻倍。

方法3：设置"不做清单"

与其列 todo，不如列 not-to-do。

比如：
- 不在上午10点前看微信
- 不在工作时开社交媒体
- 不在晚上8点后处理工作

明确你不做什么，跟明确你要做什么一样重要。

## 专注力需要训练

专注力不是天赋，是可以训练的肌肉。

一开始你只能专注15分钟。没关系。每天练习，三个月后你可以轻松专注2小时。

这2小时，是你和其他人拉开差距的关键。
