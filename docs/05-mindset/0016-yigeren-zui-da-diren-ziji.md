# 一个人最大的敌人是自己

你觉得自己最大的问题是什么？

没资金？没人脉？没技术？

都不是。

你最大的敌人是你自己。

## 阻止你前进的3个东西

第一个：等待完美时刻

你总想准备好再开始。

"等我学会这个再开始"
"等我积累更多粉丝再变现"
"等产品完善了再发布"

真相是：永远没有完美的时刻。Vigma创始人教主说过一句话："完成好过完美。先上车，再换座。"

第二个：害怕被嘲笑

你怕发出去的东西被人说不好。

但你想过没有：那些嘲笑你的人，他们自己也没做出什么名堂。

第三个：觉得自己不配

你觉得自己还不够格教别人。

但很多人需要的，不是最专业的人，而是刚好比他懂一点点的你。

## 怎么打破这个循环

方法很简单：最小可行行动。

不要计划100天。先做1天。
不要写10篇文章。先写1篇。
不要打造完美产品。先卖出第1单。

当你开始最小行动，你会发现：
- 没那么多人在意你
- 你没有想象中那么差
- 事情没那么难

## 从今天开始

找一件你一直想做但没做的事情。

24小时内做完它。

不需要准备，不需要完美。只要开始。
