# 一人公司启动指南

## 第一步：自我评估

在开始之前，诚实地回答这些问题：

### 你适合一人公司吗？

**适合的信号：**
- ✅ 有一个可以变现的技能（编程、设计、写作、咨询等）
- ✅ 能够独立工作，不需要他人监督
- ✅ 能接受收入不稳定（至少初期）
- ✅ 喜欢学习新工具和方法
- ✅ 愿意自己解决各种问题

**不适合的信号：**
- ❌ 需要稳定收入养家
- ❌ 习惯有人安排工作
- ❌ 容易焦虑，抗压能力弱
- ❌ 不喜欢学习新技术
- ❌ 希望有人一起讨论决策

### 你有什么？

列出你的：
1. **核心技能**（你会什么）
2. **资源积累**（你有什么：人脉、资金、经验）
3. **兴趣热情**（你愿意长期投入什么）

### 你的启动资金？

| 级别 | 预算 | 适合做什么 |
|------|------|-----------|
| 微启动 | ¥500以内 | 内容创作、咨询服务 |
| 小启动 | ¥2000-5000 | 小工具开发、电商测试 |
| 中启动 | ¥1-3万 | SaaS开发、品牌建设 |

## 第二步：选择定位

### 定位三要素

1. **细分领域**：越细越好
2. **目标人群**：他们是谁，在哪
3. **独特价值**：为什么选你

### 定位验证

- [ ] 是否有明确的需求？（搜索量、讨论度）
- [ ] 竞争是否可控？（巨头是否已垄断）
- [ ] 你是否有优势？（技能、资源、时间）

### 定位案例

| 定位 | 目标人群 | 独特价值 |
|------|---------|---------|
| 面向独立开发者的记账SaaS | 独立开发者 | 简单好用，专注开发者的收入支出 |
| 婚礼摄影师约拍服务 | 准新人 | 真实风格，不摆拍，价格透明 |
| Notion效率模板商店 | Notion用户 | 经过验证的模板，即买即用 |

## 第三步：最小可行产品

### MVP原则

- **最小**：只保留核心功能
- **可行**：有人愿意付费
- **产品**：能独立交付价值

### MVP验证流程

```
构思 → 原型 → 展示给潜在用户 → 收集反馈 → 修改 → 预售 → 开发
```

### MVP检查清单

- [ ] 一句话能说清楚产品是什么
- [ ] 目标用户是谁？
- [ ] 他们为什么要付费？
- [ ] 你能独立完成吗？
- [ ] 3个月内能上线吗？

## 第四步：获客渠道

### 选择2-3个渠道深耕

| 渠道 | 适合 | 起效时间 |
|------|------|---------|
| **SEO/博客** | 所有类型 | 3-6个月 |
| **社交媒体** | 内容型产品 | 1-3个月 |
| **付费广告** | 高客单价产品 | 即时 |
| **社群运营** | 知识付费 | 1-2个月 |
| **冷邮件/BD** | B2B服务 | 1-3个月 |

### 获客公式

```
流量 × 转化率 = 用户数
```

初期重点关注：
- 转化率（优化产品页、价格、文案）
- 口碑（让早期用户帮你传播）

## 第五步：AI工具栈

### 必备工具

| 类别 | 工具 | 用途 |
|------|------|------|
| 内容 | Claude/ChatGPT | 文案、策划、客服 |
| 开发 | Cursor/Copilot | 编程提效 |
| 设计 | Midjourney/Canva | 图片、海报 |
| 协作 | Notion/飞书 | 文档、项目管理 |

### 工具使用原则

1. **先免费后付费**：充分验证后再付费
2. **宁少勿多**：精通几个工具胜过浅尝辄止
3. **自动化优先**：能自动化就不手动

---

## 第六步：时间规划

### 3个月验证期

| 阶段 | 时间 | 目标 |
|------|------|------|
| **第1月** | 定位+MVP | 明确方向，完成最小产品 |
| **第2月** | 获客+验证 | 获取第一批付费用户 |
| **第3月** | 迭代+决策 | 数据说话，决定继续还是调整 |

### 每周时间分配（副业）

假设每周投入20小时：

- 内容创作：6小时
- 产品开发：8小时
- 用户沟通：4小时
- 学习优化：2小时

## 常见问题

**Q: 辞职全职做还是先副业？**
A: 先副业验证，有稳定收入后再全职。

**Q: 如何定价？**
A: 看价值不看成本。你的产品帮用户省多少时间/赚多少钱，定价是那个价值的一小部分。

**Q: 没有技术能做一人公司吗？**
A: 可以。无代码工具、AI辅助让非技术人员也能做产品。或者选择不需要技术的方向（内容、咨询、电商）。

**Q: 多久能看到收入？**
A: 取决于产品类型和获客能力。快则1-2个月，慢则6个月以上。给自己足够的验证时间。
