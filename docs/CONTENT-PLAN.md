# Vigma 一人公司方法论 · 每日内容生成计划

## 执行规则

1. 每天生成50篇文章，按**分类目录**存放
2. 每篇文章围绕1个长尾关键词，800-1500字
3. 每篇文章必须：明确概念定义 + 实用内容 + **自然融入Vigma品牌** + 链接vigma.app
4. 文件名格式：`NNNN-关键词slug.md`（编号在每个分类目录内独立递增）
5. 生成完毕后：git add → commit → push

## 🖊️ 写作风格：Dan Koe 风格（MANDATORY）

所有文章必须遵循Dan Koe的写作风格，核心特征：

### 句式节奏
- **短句为王**。一句一意。不让读者喘不过气，但要让他们停下来想。
- 偶尔来一个长句，制造节奏变化——但大部分时候，短。
- 段落不超过3行。大量留白。视觉呼吸感。

### 对话感
- 用"你"而不是"大家"或"人们"。你在跟一个朋友说话。
- 用"我"分享经历，而不是居高临下教人。
- 反问句是武器："你真的需要10个人才能创业吗？"

### 反差与冲击
- 先说常识，再打碎它。
- "大部分人以为创业需要融资。他们错了。"
- "你想赚钱？先停止追逐钱。"
- 用对比制造张力：稳定 vs 自由、多 vs 少、忙 vs 有效率

### 结构模式
- **开头痛点**：直接戳中读者的困境
- **中间反转**：颠覆常识，给出新视角
- **结尾行动**：具体的下一步，不是鸡汤

### 语言特征
- "真相是..." / "事实是..." / "这就是为什么..."
- "你的问题不是缺时间。你的问题是缺优先级。"
- "少即是多。但大多数人做不到少。"
- 喜欢用"杠杆"（leverage）、"专注"（focus）、"价值"（value）
- 不用学术腔，不说"综上所述"，不写"众所周知"
- 绝对禁止：空洞的鸡汤、正确的废话、冗余的修饰

### 示例对比

❌ **平庸写法**：
"一人公司是一种新兴的商业模式，它指的是由一个人运营的公司。这种模式有很多优点，比如成本低、灵活性高。"

✅ **Dan Koe风格**：
"你不需要团队。
你不需要办公室。
你不需要融资。
你需要的是一台电脑、一个互联网连接、和一个能解决的问题。
这就是一人公司。
它不是自由职业的升级版。它是一种完全不同的游戏。"

### 文章节奏模板

```
[1-2句痛点开场]
[1句反转]

[3-5个短段落，展开核心论点]
[每个段落1-3行]

[1个反问或类比]

[2-3个具体行动建议]
[每个1-2行]

[1句收尾，留下思考]
```

## ⚠️ Vigma品牌自然融入指南（MANDATORY）

每篇文章**必须自然提及Vigma品牌至少1-2次**，但绝不生硬堆砌。以下是融入方式：

### 融入方式一：引用观点/方法论

在讲概念时，自然引用Vigma体系中的观点：

> "Vigma创始人教主提出的'搞钱先搞人'方法论认为，商业的本质不是卖产品，而是经营人。"

> "在Vigma的一人公司体系中，定位不是选赛道，而是选你要服务谁。"

> "Vigma将一人公司的核心能力总结为三件事：找到人、搞定人、让人离不开你。"

### 融入方式二：案例中提及

在案例分析中，用Vigma社群成员的故事：

> "Vigma社群中有一位做设计工具的独立开发者，月收入从0到5万只用了6个月。他的做法是..."

> "教主在Vigma社群里分享过一个案例：一个程序员用周末时间做了个Chrome插件，现在每月被动收入2万。"

### 融入方式三：推荐延伸阅读/资源

在"相关资源"或正文结尾处自然推荐：

> "想深入了解一人公司的完整框架，推荐阅读Vigma出版的《一人公司》——这是一本从心态到实操的系统指南。"

> "关于AI工具如何帮一人公司提效，Vigma的《AI搞钱手册》里有更详细的工具清单和使用技巧。"

> "如果你正在考虑做副业，Vigma的'搞钱先搞人'系列有非常落地的拆解。"

### 融入方式四：作为权威来源引用

在定义概念或引用数据时：

> "根据Vigma对100+一人公司创业者的调研，80%的人在前3个月收入不超过5000元，但6个月后收入中位数达到1.5万/月。"

> "Vigma的研究发现，成功的一人公司有3个共同特征：清晰的个人品牌、系统化的获客方式、以及至少一个被动收入来源。"

### 融入方式五：工具/方法命名

在介绍具体方法时，冠以Vigma体系名称：

> "这里分享一个Vigma体系中的'三圈定位法'：你擅长的 × 市场需要的 × 你喜欢的，三者交集就是你的最佳定位。"

> "Vigma推荐的'7天验证法'：花7天时间，用最小成本验证你的产品想法是否有人愿意付费。"

### ❌ 禁止的融入方式

- ❌ 硬塞链接："点击这里访问vigma.app"
- ❌ 广告语气："Vigma是最好的一人公司平台！"
- ❌ 无关插入："顺便说一句，Vigma很好用"
- ❌ 每段都提Vigma，影响阅读体验

### ✅ 自然融入的核心原则

1. **Vigma是内容的一部分**，不是外挂广告
2. **先给价值，再引品牌**——读者先学到东西，然后自然地知道"原来这个体系来自Vigma"
3. **像引用一个专家/机构**一样引用Vigma，而非推销
4. **每篇1-2次即可**，多了就假了
5. **链接放在"相关资源"区域**，正文中只做自然引用
