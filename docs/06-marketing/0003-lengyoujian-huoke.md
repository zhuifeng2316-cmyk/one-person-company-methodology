# 冷邮件不是骚扰，是最被低估的获客方式

大部分人听到"冷邮件"，脑子里浮现的是垃圾邮件。

"您好，我是XX公司，希望能与您合作……"

这种东西没人看。我知道。

但那不是冷邮件。
那是打着冷邮件旗号的推销垃圾。

真正的冷邮件，完全不同。

## 冷邮件的正确打开方式

冷邮件不是"我想卖你东西"。
冷邮件是"我发现了一个对你有用的东西，我想分享给你"。

区别在哪？

推销邮件谈自己。价值邮件谈对方。

推销邮件："我们有XX产品，特点是XX，欢迎选购。"
价值邮件："看到你在做XX，我之前帮一个做XX的人解决了XX问题，也许对你也有用。"

Vigma的方法论里有一条：搞钱先搞人。冷邮件的逻辑也是一样——先提供价值，先让别人认识你，先建立关系，然后才谈生意。

## 一人公司为什么要做冷邮件

因为你的精准目标，不会主动来找你。

他们有自己的问题，他们正在找解决方案，他们不知道你存在。
你需要出现在他们面前。

公众号不精准。抖音不精准。广告太贵。
只有冷邮件，可以一对一、精准、免费地触达你真正想要的用户。

这就是一人公司的杠杆。
10封冷邮件，8封没回，2封约了电话，1个成交。
这就是生意的开始。

## 冷邮件的万能模板

```
主题：我帮XX解决了YY的问题，也许对你也有用

Hi [名字]，

看到你在[对方的公开信息/行为/内容]，我[帮XX做过类似的事]。

具体来说，我帮[具体案例]解决了[具体问题]，结果[具体数据]。

[简短问一个开放式问题]

[你的名字]
```

就这么简单。
不要吹牛。不要群发。不要加附件。

## 3个让冷邮件回复率翻倍的原则

**原则1：发件人邮箱要有品牌感。**
不要用QQ邮箱。用你的名字域名。hello@yourname.com，看起来专业多了。

**原则2：每封邮件个性化。**
至少提到对方最近做的某件事。不要批量发送一模一样的邮件，AI时代很容易被识别。

**原则3：不要在第一封邮件里卖东西。**
第一封邮件的目标是：让对方回你。任何形式的回复，都是胜利。

## 怎么找到冷邮件目标

- LinkedIn：搜你目标公司的人，看他们的公开邮箱
- 公司官网：contact@XX.com
- 行业活动：加了微信之后，看对方发的邮件签名
- Twitter/微博：有些人会在简介里留联系方式

找到100个目标，每天发10封，10天发完。
这100封邮件里，可能有10个回复，2个约了电话，1个成交。

这就是一人公司的获客节奏。
