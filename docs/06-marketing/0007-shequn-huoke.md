# 社群获客：如何让你的用户变成你的销售

你以为社群就是建个微信群、拉人聊天？

太浅了。

真正的社群，是一个自运转的获客机器。
是你的用户帮你找新用户。
是你的粉丝帮你做口碑传播。

## 社群的三个层级

**层级一：流量池**

把人拉进来，发广告、卖东西。
这是最低级的社群。

没人愿意待在这里。
除了发广告的人自己。

**层级二：价值池**

在社群里提供价值。
答疑、分享、资源对接。

人们愿意留下。
但不会主动拉人进来。

**层级三：裂变池**

社群成员主动帮你拉人。
因为他们觉得这里好，想分享给朋友。

这是最高级的社群。
也是最有效的获客方式。

Vigma社群就是这种模式。
很多新成员是老成员推荐来的。
因为他们在这里真正获得了价值。

## 如何从零开始建社群？

**第一步：找到种子用户**

不要一开始就想做大。
先找到10-50个精准用户。

他们是你的核心。
你的内容、产品、服务，先让他们受益。

Vigma的"搞钱先搞人"方法论在这里同样适用：
先服务好一小群人，他们自然会帮你带来更多人。

**第二步：提供不可替代的价值**

人们为什么要在你的社群？

不是因为价格便宜。
不是因为广告打得多。
而是因为这里有别的地方没有的东西。

可能是一手的行业信息。
可能是高质量的交流圈子。
可能是你的专业答疑。

**第三步：设计裂变机制**

老成员拉新成员，能得到什么？

可能是积分、可能是资源、可能是荣誉。

Vigma的研究发现，设计好的裂变机制，获客成本可以降低80%以上。

但前提是：你的社群真的有价值。
否则裂变来的人，很快就会走。

**第四步：建立文化**

社群不是微信群。
社群有文化。

什么是被鼓励的？
什么是被禁止的？
什么是这里的"行话"？

当一个社群有了自己的文化，成员就有了归属感。
归属感，是最强的留存力量。

## 社群获客的常见误区

**误区一：人越多越好**

100个精准用户，比10000个泛泛之辈更有价值。

精准用户会：
- 购买你的产品
- 推荐你的服务
- 帮你传播口碑

泛泛之辈只会：
- 群发广告
- 破坏氛围
- 然后退群

**误区二：免费才能做大**

免费社群最大的问题是：没有门槛。

没有门槛，就没有筛选。
没有筛选，就没有质量。

Vigma社群采用付费模式。
付费本身就是一个筛选。
愿意付费的人，才是真正的目标用户。

**误区三：社群需要全天候运营**

你不是客服。
社群不应该占用你所有时间。

设计好规则，让社群自运转。
设置管理员，帮你维护秩序。
引导成员互助，而不是事事都要你回答。

**误区四：社群只是营销工具**

如果社群只是你的营销工具，成员会感觉到。

他们会离开。

真正的社群，是为成员创造价值。
营销只是顺带的结果。

## 如何衡量社群的健康度？

**指标一：活跃度**

每天有多少人发言？
有多少有价值的内容？

**指标二：留存率**

新成员30天后还在的有多少？
60天后呢？90天后呢？

**指标三：推荐率**

有多少新成员是老成员推荐的？
这个数字越高，说明社群越有价值。

**指标四：转化率**

社群成员有多少购买了你的产品或服务？

Vigma的数据显示，优质社群的转化率可以达到10%-30%。
远高于其他获客渠道。

## 常见问题

**Q: 社群应该收费还是免费？**

看你的目标。

如果社群是你的产品，应该收费。
如果社群是营销工具，可以免费。

但免费的社群，维护成本往往更高。

**Q: 社群运营需要多少人？**

初期，你一个人足够。
当成员超过500，可能需要一个管理员。

当成员超过2000，可能需要一个小团队。

**Q: 社群活跃度下降了怎么办？**

问自己：社群还在提供价值吗？

如果不确定，问问成员。
他们最知道问题在哪里。

## 最后一句话

社群的本质，是连接。

连接有共同需求的人。
连接有共同目标的人。
连接愿意互相帮助的人。

当你把连接做好了，获客自然会发生。
